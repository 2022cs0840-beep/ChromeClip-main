# ChromeClip

ChromeClip is a Chrome extension that listens to copy events and stores copied text in Chrome's sync storage. It also provides a shortcut to quickly copy selected text.

## Features
- **Automatic Clipboard Tracking**: Captures and stores copied text.
- **Sync Storage**: Saves copied text across different Chrome devices.
- **Quick Copy Shortcut**: Allows users to copy selected text instantly.
- **User-Friendly UI**: Displays stored clipboard history.

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/your-username/chromeclip.git
   cd chromeclip
   ```
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** and select the project folder.

## Usage
- Copy any text, and ChromeClip will store it automatically.
- Open the extension popup to view your copied history.
- Use the shortcut key (configurable) to copy selected text instantly.



## Technologies Used
- **JavaScript** for core logic
- **Chrome Extensions API** for clipboard access and storage
- **Manifest V3** for security and performance
- **HTML & CSS** for UI design
